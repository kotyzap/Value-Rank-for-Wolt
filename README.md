# Value-Rank-for-Wolt
On any Wolt category page it scans the items once, then marks each card with its price per unit and value rank — 🥇🥈🥉 for the 3 cheapest of each unit and rank/total (e.g. 15/30) for the rest. Nothing is reordered, so add-to-cart and the info modal keep working. Category pages only (URLs with /items/).
